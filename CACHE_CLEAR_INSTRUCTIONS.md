# Complete Cache Clearing Instructions

## Chrome/Edge/Brave
1. Press `Ctrl+Shift+Delete` (or `Cmd+Shift+Delete` on Mac)
2. Select "All time"
3. Check: Cookies, Cache, Cached images/files
4. Click "Clear data"
5. Close DevTools and Reload: `Ctrl+Shift+R`

## Firefox
1. Press `Ctrl+Shift+Delete`
2. Select "Everything"
3. Click "Clear Now"
4. Hard reload: `Ctrl+Shift+R`

## Safari
1. Develop → Empty Caches
2. Command+R to reload

## Service Worker Removal
1. Open DevTools → Application
2. Service Workers tab
3. Click "Unregister" on all service workers
4. Go to Storage tab
5. Clear all site data for the domain
